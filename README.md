**Inventory-and-Sales-Management-System-for-Elkim**
This project presents the design and development of a web-based Inventory and Sales Management System for Elkim Chemistry, a pharmaceutical business operating multiple branches within Nairobi County.
The system is designed to replace manual record-keeping methods such as paper files and spreadsheets by providing a centralized platform for managing inventory, recording sales, and monitoring business performance in real time.
**Objectives**
Automate inventory and sales management processes
Reduce human errors and data inconsistencies
Improve transparency and accountability
Enable centralized monitoring of multiple branches
Support data-driven decision-making
**Technologies used**
PHP (Server-side scripting)
MySQL (Database)
HTML (Structure)
CSS (Styling)
JavaScript (Client-side interaction)
XAMPP (Local server environment)
**System features**
User authentication (Login system)
Product and category management
Customer and supplier management
Sales recording and tracking
Automatic stock updates
Profit calculation
Dashboard with charts and reports
Low-stock alerts
Expiry date tracking
Data export to Excel

Documentation (Proposal, SAD, Final Report)

N.B. The system is currently running on a local server and can be accessed locally thorugh the process below
**How to run the system locally**
Install XAMPP
Start Apache and MySQL
Copy the project folder into:
C:/xampp/htdocs/
Open your browser and go to:
http://localhost/your-project-folder
Open phpMyAdmin:
http://localhost/phpmyadmin
Create a database
Import the provided .sql file
Run the system in your browser
For Elkim, the owner will purchase a domain name for hosting the system in a web server and go live online whereby the running server will enable Elkim to run the system and store their database.
